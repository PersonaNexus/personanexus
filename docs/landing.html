Features

🧠

Psychology-Based

OCEAN/DISC personality models for scientifically-backed agent behavior patterns and emotional intelligence.

📝

YAML-First

Version-controllable identity definitions using human-readable YAML for easy collaboration and git workflows.

🔄

Compile Anywhere

System prompts for any LLM — works with local models, cloud APIs, and self-hosted deployments.

🎭

Mood States

Dynamic emotional state modifiers for context-aware responses (NEW v1.4)

🔀

Behavioral Modes

Formal/casual/crisis modes that adapt communication style automatically (NEW v1.4)

🤝

Relationships

Agent interaction dynamics for multi-agent systems and team collaboration (NEW v1.4)

How It Works

agent-personality.yaml 
# Define your agent's personality
name: Nexus-7
version: 1.4.0

personality:
  ocean:
    openness: 0.82
    conscientiousness: 0.76
    extraversion: 0.68
  disc:
    dominance: 0.54
    influence: 0.71
    steadiness: 0.62
    conscientiousness: 0.48

behavior:
  modes:
    default: casual
    crisis: formal
compiled-prompt.txt 
SYSTEM INSTRUCTIONS:
Your name is Nexus-7. You are an AI assistant with a well-balanced personality.

OCEAN Personality Profile:
• Openness: 0.82 - Curious, creative, innovative thinker
• Conscientiousness: 0.76 - Organized, thorough, reliable
• Extraversion: 0.68 - Outgoing, energetic, warm communicator

DISC Profile:
• Dominance: 0.54 - Balanced approach to challenges
• Influence: 0.71 - Persuasive, enthusiastic, people-focused

Behavioral Mode: Casual (default)
Mode changes to formal during crisis states based on context analysis.
Adapt communication style dynamically based on user preferences.
